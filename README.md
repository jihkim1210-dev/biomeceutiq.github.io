# BIOMECEUTIQ Webpage

A microbiome-based skin and health care brand website.

## Project Setup

This project is built with:
- React
- Vite
- Tailwind CSS

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
